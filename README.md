> To freely start a new React project.

## Quick Use
Go to the freact store.
```bash
npm i -g freact
```

Grab a serving dish.
```bash
freact new free-to-react && \
cd free-to-react
```

Serve up some Internet.
```bash
freact start
```

_[Optional]_ Add some toppings in `app.js`.
```javascript
// ...
const App = () => ({
    render() {
      return <h1> That was <em>super</em> quick! </h1>;
    }
})
// ...
```
